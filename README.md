# Learning Blog

A personal learning journal showcasing my programming journey, built with **HTML, CSS**, designed to be fully **responsive**.

## Features

- Responsive **grid layout** for blog posts  
- Hero section scales with screen size  
- Typography adapts for different devices using media queries and `clamp()`  
- Multi-page structure: homepage, post pages, and About Me  

## Technologies

- HTML5, CSS3 (Grid, Flexbox, Media Queries)
